昨晚把老头环学习版启动成功了，感慨下几个注意点以及碰到的坑

1. 到手直接拆机换SSD，建议用1字改锥（小米那个1.5），电源胶带没有黏在电源IC上是浮空的，建议缠绕下以防黏在芯片上悲剧，还原要注意把保护罩扣好，贴吧烧电源IC的应该都是IC和散热垫没有直接接触造成的，不拆机也有概率会出现黑色胶带被临时工黏在芯片上的情况

2. SteamOS正版很省心，操作人性化，功能一看就懂。 像我这样到手Steam游戏是0（免费的也没有）的估计也没谁了吧

就跟PC一样，先分区（ventoy + gparted），然后用ventoy安装Windows的ISO。分区过程很慢，不是满电开搞的，一度担心电量撑不住。装系统就很常规了，装完之后要手动安装一堆官网的驱动。

2. XGP

主要问题是手柄支持，主要有两种解决方案：

- 走GlosSI，可以把XGP（UWP）的应用添加到Steam里，也可以添加exe和epic，然后通过Steam启动，就能用手柄了，[https://github.com/Alia5/GlosSI](https://github.com/Alia5/GlosSI)

- Steam Deck Windows Usermode Driver，开启后能让系统把SD手柄识别为360手柄，然后就能直接用了，[https://github.com/mKenfenheuer/ ... ows-usermode-driver](https://github.com/mKenfenheuer/steam-deck-windows-usermode-driver)

试了下Pentiment，不知道为什么画面开不了全屏，还没进一步琢磨

3. 网游1：冒险岛MapleStory国际服

小时候玩的游戏，前段时间职业大重做就捡起来玩了玩，国际服可以从Steam启动，感觉刷刷刷还挺适合SD（可以躺着肝）奈何Steam OS下Proton兼容不了

Windows下通过Steam安装，编辑了一通键位映射，结果进游戏发现键位映射完全不起作用。。然后搜了一圈，说是要Steam以管理员身份启动，试了下果然可以躺着爽玩了

4. 网游2：FF14国服

其实我基本没玩过，只是想看看效果。用GlosSI会因为没有一个好用的直接启动的exe所以用不了，所以是用Steam Deck Windows Usermode Driver解决的，里面有个鼠标键盘选项要禁用掉，进去后和PC上拿手柄玩体验完全一致，画面选项要开全屏无边框
